# 📰 NewsHub - News Aggregator

## Overview

NewsHub is a responsive web application that aggregates and displays news articles in a clean and user-friendly interface. It allows users to browse articles by category, search for specific topics, manage favorites, and switch between multiple languages.

## Features

- 🔍 Search news articles
- 📰 Browse by category
- ❤️ Save favorite articles
- 🌍 Multilingual interface
- 📄 Pagination
- 📱 Responsive design
- 🎨 Modern and intuitive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## Project Structure

```
NewsHub/

│── assets/
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/NewsHub.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
